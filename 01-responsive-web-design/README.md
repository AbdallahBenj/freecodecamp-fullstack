# 🌐 Responsive Web Design Certification

freeCodeCamp – **Full Stack Developer Certification (Responsive Web Design Path)**

This repository contains my work, exercises, and projects from the **Responsive Web Design Certification** within the new **freeCodeCamp Full-Stack Developer Curriculum**.

Throughout this learning path, I built several hands-on projects to learn and practice the fundamentals of **HTML**, **CSS**, modern layout techniques, **accessibility**, and **responsive design**.

The goal of this path is to create real, clean, accessible, and responsive webpages using HTML and CSS — applying core concepts and developing the skills needed to build professional, user-friendly interfaces.

---

## ⭐ About the Certification

🔹 In this certification, I practiced building real UI components and layouts through hands-on projects.
The curriculum covered:

🔹 HTML fundamentals

🔹 CSS styling and selectors

🔹 Layout techniques (Flexbox and Grid)

🔹 Accessibility best practices

🔹 Responsive design for all screen sizes

---

### 📘 What I Learned

🔹 Core Web Foundations

🔹 Writing semantic HTML

🔹 Styling pages with CSS

🔹 Using selectors, units, colors, and spacing

🔹 Working with images, forms, and links

### Modern Techniques

🔹 CSS Variables

🔹 CSS positioning

🔹 Flexbox for responsive layout

🔹 CSS Grid for advanced layouts

🔹 Accessibility (ARIA, labels, structure)
